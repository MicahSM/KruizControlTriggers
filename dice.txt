OnCommand e 4 !roll
  Function 'var arr = [after].split("d"); return {quantity: arr[0], magnitude: arr[1]}'
  Function 'return {multitude: [quantity]*[magnitude]}'
  Random Number {quantity} {multitude}
  Chat Send "{user} rolled {number} 🎲"
  exit
